# D2532C SOP System — P&K New Energy Service Center

ระบบ SOP สองภาษา (ไทย/อังกฤษ) สำหรับเครื่อง **SUNKKO D2532C** — เครื่องวิเคราะห์และปรับสมดุลแบตเตอรี่ EV พร้อมฐานข้อมูลแบตเตอรี่แรงดันสูง (HV) หลายยี่ห้อ และระบบสร้างรายงานตรวจสอบระดับมาตรฐาน Lab

Bilingual (Thai/English) SOP system for the **SUNKKO D2532C** EV battery pack equalizer/analyzer, with a multi-brand HV battery database and a lab-grade inspection report generator.

## โครงสร้างโปรเจกต์ / Project structure

```
neta-d2532c-sop/
├── index.html          # Main entry point (shell + CDN library imports)
├── css/
│   └── style.css       # Full stylesheet (light theme, laptop layout, print/A4, report doc)
├── js/
│   ├── data.js         # Reference data: HV battery DB, parameters, SOP, decision table, import/export logic
│   ├── i18n.js          # Thai/English UI strings + language switch
│   ├── export.js       # Excel (.xlsx) import/export and PDF export functions
│   └── app.js           # Routing, rendering, QC form, battery DB, inspection report, ΔV calculator
└── README.md
```

No build step required — pure HTML/CSS/JavaScript (vanilla). Deployable directly to GitHub Pages, or opened locally via any static file server.

## สิ่งที่อัปเดตในเวอร์ชันนี้ / What's new in this version

- **แก้ไขการสลับภาษา** — เปลี่ยนจากปุ่มสลับเดี่ยว (กดแล้วงง) เป็นปุ่มเลือกแบบ segmented control `[ไทย] [EN]` ที่เห็นชัดว่ากำลังใช้ภาษาไหนอยู่ ลดความสับสนตอนใช้งานจริง — ตรวจสอบข้อความในระบบทั้งหมดให้ตรงกับภาษาที่เลือกแล้ว ไม่มีข้อความค้างภาษาอื่นปน
- **หน้า Parameter Settings ขยายครบทุกยี่ห้อ + ค้นหา/กรองได้** — จากเดิมมีแค่ NETA 5 รุ่น ตอนนี้ครบทั้ง 15 รุ่น 3 ยี่ห้อ (NETA/MG/GAC AION) โดยแถวของ NETA เป็นค่าที่ยืนยันจากคู่มือเครื่อง D2532C ส่วนยี่ห้ออื่นเป็น "ค่าอ้างอิงคำนวณ" จากเคมีแบตเตอรี่ (มีป้ายกำกับชัดเจนแยกสองประเภท ควรตรวจสอบกับ BMS จริงก่อนใช้งาน) มีช่องค้นหาแบบพิมพ์กรองสด และแท็บกรองตามยี่ห้อ — และนำเข้า/ปรับปรุงฐานข้อมูลพารามิเตอร์เองผ่านไฟล์ Excel ได้เช่นเดียวกับฐานข้อมูลแบตเตอรี่
- **ฐานข้อมูลแบตเตอรี่ HV ขยายใหญ่ขึ้น** — จากเดิม NETA 4 รุ่น เป็น **15 รุ่น 3 ยี่ห้อ** (NETA, MG, GAC AION) พร้อมแก้ไขค่าความจุที่ขาดหายไป (เช่น NETA V1 Gen2) และเพิ่มรุ่นใหม่ (NETA X Smart) — มีแท็บกรองตามยี่ห้อ และคอลัมน์ผู้ผลิตแบตเตอรี่/หมายเหตุ
- **นำเข้าฐานข้อมูลผ่าน Excel** — ปุ่ม "ดาวน์โหลดเทมเพลต Excel" ให้ไฟล์ตัวอย่างพร้อมหัวตาราง กรอกข้อมูลแล้วนำเข้ากลับเข้าระบบได้ทันที ระบบจะจับคู่ตามชื่อรุ่นรถ (อัปเดตรุ่นเดิม/เพิ่มรุ่นใหม่อัตโนมัติ) ใช้ได้ทั้งฐานข้อมูลแบตเตอรี่และฐานข้อมูลพารามิเตอร์ ข้อมูลที่นำเข้าจะถูกบันทึกไว้ในเบราว์เซอร์ (localStorage) และมีปุ่ม "คืนค่าเริ่มต้น" เพื่อล้างข้อมูลที่นำเข้าเอง
- **รายงานตรวจสอบ (Inspection Report)** — หน้าใหม่ที่รวม (1) ผลตรวจ QC ของคัน/ครั้งที่เลือก (2) ข้อมูลอ้างอิงแบตเตอรี่ NETA ทั้งหมด และ (3) ตารางตัดสินใจ ΔV เข้าเป็นเอกสารเดียว จัดหน้าแบบรายงานผลตรวจสอบห้องแล็บ (โลโก้/หัวบริษัท, เลขที่เอกสาร, ตารางข้อมูล, แถบสรุปผลผ่าน/ไม่ผ่าน, ช่องลงลายเซ็นช่างเทคนิคและผู้อนุมัติ) พิมพ์ A4 หรือ Export PDF ได้ในคลิกเดียว
- **Layout ใหม่** — ธีมสว่าง พื้นขาว เมนูแนวนอนด้านบน เหมาะกับหน้าจอ laptop (ต่อเนื่องจากเวอร์ชันก่อนหน้า)

## ฟีเจอร์ทั้งหมด / All features

- **สองภาษา (TH/EN)** — สลับได้ทันทีทุกหน้าด้วยปุ่มเลือกภาษาแบบ segmented control
- **Dashboard** — สรุปภาพรวม จำนวนรุ่นแบตเตอรี่ในฐานข้อมูล จำนวนบันทึก QC ทางลัดไปยังส่วนต่างๆ
- **SOP Steps** — 8 ขั้นตอนมาตรฐาน + ข้อควรระวังความปลอดภัย + วิธีใช้เครื่องละเอียด 12 ขั้นตอน
- **Parameters** — ตารางพารามิเตอร์การตั้งค่าเครื่อง D2532C ครบทั้ง 15 รุ่น 3 ยี่ห้อ พร้อมค้นหา/กรองตามยี่ห้อ ป้ายกำกับแหล่งข้อมูล (ยืนยันจากคู่มือ/ค่าอ้างอิงคำนวณ) glossary และตารางตัดสินใจ ΔV — นำเข้า/ปรับปรุงได้ผ่าน Excel
- **Battery Database (HV)** — ข้อมูลจำเพาะแบตเตอรี่ 15 รุ่น 3 ยี่ห้อ กรองตามยี่ห้อ นำเข้า/ส่งออกผ่าน Excel
- **QC Form** — กรอกค่าก่อน/หลังบาลานซ์ คำนวณ ΔV และคำแนะนำอัตโนมัติ บันทึกลง localStorage
- **Inspection Report** — สร้างรายงานตรวจสอบระดับมืออาชีพจากบันทึก QC + ฐานข้อมูลแบตเตอรี่ + ตารางตัดสินใจ พิมพ์ A4 / Export PDF
- **ΔV Calculator** — คำนวณด่วน เทียบกับเกณฑ์ Stop ΔV ของแต่ละรุ่น
- **Export Excel (.xlsx)** — บันทึก QC, ข้อมูลอ้างอิงทั้งหมด, และเทมเพลตนำเข้าฐานข้อมูล (ผ่าน SheetJS)
- **Export PDF / พิมพ์ A4** — ทุกหน้าหลัก รวมถึงรายงานตรวจสอบ (ผ่าน jsPDF + html2canvas และ CSS `@media print`)
- **Responsive UI** — ใช้งานได้ทั้งเดสก์ท็อป/laptop และมือถือ (เมนูแบบ drawer บนหน้าจอเล็ก)

## การใช้งาน / Usage

1. เปิด `index.html` ในเบราว์เซอร์ (ต้องเชื่อมต่ออินเทอร์เน็ตเพื่อโหลดฟอนต์ Google Fonts และไลบรารี Excel/PDF)
2. หรือ deploy ทั้งโฟลเดอร์ไปยัง GitHub Pages / เว็บโฮสติ้งใดก็ได้
3. ข้อมูล QC และฐานข้อมูลแบตเตอรี่ที่นำเข้าเองจะเก็บไว้ใน localStorage ของเบราว์เซอร์นั้นๆ (ต่ออุปกรณ์) — ใช้ปุ่ม "ส่งออก Excel" เพื่อสำรอง/แชร์ข้อมูล

### วิธีอัปเดตฐานข้อมูลแบตเตอรี่ / Updating the battery database

1. ไปที่หน้า **ฐานข้อมูลแบตเตอรี่** → กด **ดาวน์โหลดเทมเพลต Excel**
2. กรอกข้อมูลตามคอลัมน์: Brand, Vehicle Model, Chemistry, Strings, Modules, Nominal Voltage (V), Capacity (Ah), Energy (kWh), Battery Supplier / Note
3. กด **นำเข้าไฟล์ Excel** แล้วเลือกไฟล์ที่กรอกแล้ว — ระบบจะอัปเดตรุ่นที่มีชื่อตรงกัน และเพิ่มรุ่นใหม่ที่ไม่เคยมี
4. กด **คืนค่าเริ่มต้น** หากต้องการล้างข้อมูลที่นำเข้าเองและกลับไปใช้ฐานข้อมูลเริ่มต้น

### วิธีอัปเดตฐานข้อมูลพารามิเตอร์ / Updating the parameter database

1. ไปที่หน้า **พารามิเตอร์** → กด **ดาวน์โหลดเทมเพลต Excel**
2. กรอกข้อมูลตามคอลัมน์: Vehicle Model, Battery Type, Strings, Charge Current (A), Balance Normal (A), Balance Heavy (A), Mode, Target Cell (V), Charge Voltage (V), Stop Delta V, Source/Note (ใส่ "verified" หากยืนยันจากคู่มือจริง หรือ "estimated" หากเป็นค่าอ้างอิง)
3. กด **นำเข้าไฟล์ Excel** — ระบบจับคู่ตามชื่อรุ่นรถเช่นเดียวกับฐานข้อมูลแบตเตอรี่
4. ใช้ช่องค้นหาด้านบนตารางเพื่อกรองรุ่นรถ หรือแท็บยี่ห้อเพื่อดูเฉพาะยี่ห้อที่ต้องการ

### วิธีสร้างรายงานตรวจสอบ / Generating an inspection report

1. ไปที่หน้า **รายงานตรวจสอบ**
2. เลือกบันทึก QC ที่บันทึกไว้แล้ว (ระบบจะดึงข้อมูลมาเติมอัตโนมัติ) หรือกรอกข้อมูลเองด้านล่าง
3. ตรวจสอบตัวอย่างรายงานที่แสดงด้านล่าง แล้วกด **พิมพ์ (A4)** หรือ **ส่งออก PDF**

## แหล่งข้อมูล / Data sources

- `D2532C_Manual.pdf` — คู่มือผู้ใช้เครื่อง SUNKKO D2532C (สเปค, การต่อสาย, การตั้งค่าพารามิเตอร์, error code)
- `NETA_D2532C_SOP_Standard.xlsx` — มาตรฐาน SOP ของศูนย์บริการ (Battery_List, Parameter_Setting, SOP, Decision, Machine_Operation)
- `EV_Battery_Specs_Thailand.xlsx` — ฐานข้อมูลแบตเตอรี่ HV เปรียบเทียบ NETA / MG / GAC AION ที่จำหน่ายในไทย

## หมายเหตุ / Note

ระบบนี้เป็นเครื่องมือช่วยอ้างอิงสำหรับช่างเทคนิคเท่านั้น ไม่ได้แทนที่วิจารณญาณของช่างหรือคู่มือฉบับเต็มของเครื่อง D2532C พารามิเตอร์การตั้งค่าเครื่อง (หน้า Parameters) เฉพาะแถวของ **NETA** เท่านั้นที่ยืนยันจากคู่มือเครื่อง D2532C จริง ส่วนยี่ห้ออื่น (MG, GAC AION) เป็น **ค่าอ้างอิงคำนวณ** จากเคมีแบตเตอรี่และจำนวนสตริง (มีป้ายกำกับ "ค่าอ้างอิงคำนวณ" ในตารางชัดเจน) — ควรตรวจสอบกับข้อมูล BMS จริงของรถก่อนใช้งานเครื่อง D2532C กับแพ็คที่ไม่ใช่ NETA เสมอ
